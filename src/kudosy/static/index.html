Athleten-Verwaltung

Pro Athlet entscheidest du: Deny überspringt alle Aktivitäten, Allow gibt immer Kudos (übersteuert Distanz-/Zeit-Regeln).

    Kudo-Regeln

    Pauschalregeln (alle Sportarten)

    Gilt für alle Sportarten ohne eigene Regel.
    Auf 0 setzen, um den jeweiligen Catch-all zu deaktivieren. Pro-Sportart-Regeln unten überschreiben diese Werte.

    ℹ️ Kudos werden nur vergeben, wenn für die Sportart mindestens eine Distanz- oder Dauer-Regel aktiv ist (Sportart, Kategorie oder Pauschalregel). Erlaubte Athleten und Name-Treffer sind ausgenommen.

    Kudo-Regeln: Min. Distanz (pro Sportart)

    Aktivitäten unter dieser Distanz (km) werden übersprungen. Überschreibt die Pauschalregel für diese Sportart. Leer lassen = Pauschalregel gilt.

    Sportart Min. km

    Kudo-Regeln: Min. Dauer (pro Sportart)

    Aktivitäten unter dieser Dauer (Minuten) werden übersprungen. Überschreibt die Pauschalregel für diese Sportart. Leer lassen = Pauschalregel gilt.

    Sportart Min. min

    Auto-Kudos nach Aktivitätsname

    Regex-Muster — passende Aktivitäten erhalten immer Kudos, unabhängig von Distanz/Dauer.

      Automation

      Scheduler

      Minimum 5 Minuten. Empfohlen: 60 (stündlich).
      Zufälliger Versatz zum Intervall (±). Macht den Takt unvorhersehbar. Empfohlen: 15.

      Ausführungsmodus

      Wenn aktiv, sendet der Scheduler keine echten Kudos.

      Menschliche Timing-Delays

      Zwischen den einzelnen Kudos-Anfragen wird eine zufällige Pause eingelegt — verhindert maschinenhaftes Verhalten.

      Minimum 0. Empfohlen: 3.
      Muss ≥ Min. sein. Empfohlen: 25.
      Kudos werden in zufälliger Reihenfolge vergeben (nur im Nicht-Dry-Run).

      Ruhezeiten

      Definiere wann Kudos gegeben werden dürfen. Grüne Zellen = erlaubt. Klicke auf eine Zelle um sie umzuschalten, oder auf eine Zeilen-/Spaltenüberschrift um die gesamte Zeile/Spalte zu toggeln.

      IANA-Zeitzone, z. B. Europe/Berlin, America/New_York

      Benachrichtigungen

      HTTP-POST-Endpunkt (ntfy, Telegram-Bot, beliebige API). Leer lassen um Benachrichtigungen zu deaktivieren.
      Sendet eine Zusammenfassung (Kudos gegeben, Aktivitäten geprüft).
      Sofortiger Hinweis wenn der Strava Session-Cookie abgelaufen ist.
